# gatsby-plugin-proof

> Add Proof to your Gatsby site

## Install Your Flavor

`npm install --save gatsby-plugin-proof`

`yarn add gatsby-plugin-proof`

## Setup

In your `gatsby-config.js` file:

```javascript
plugins: [
  {
    resolve: `gatsby-plugin-proof`,
    options: {
      pixelId: `PROOF_PIXEL_ID`
    }
  }
];
```
